## Up and running

Clone the repository.

```bash
$ git clone https://github.com/Chatatata/sentinel.git
```

Install the dependencies.

- with [yarn](https://yarnpkg.com/)

```bash
$ yarn
```

- with [npm](https://npmjs.com)

```bash
$ npm install
```

Run the application.

- with [yarn](https://yarnpkg.com/)

```bash
$ yarn start
```

- with [npm](https://npmjs.com)

```bash
$ npm start
```
